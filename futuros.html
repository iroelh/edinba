Este sitio reúne dos herramientas clave para explorar futuros desde la creatividad y el diseño: el Cono de plausibilidad y el uso de la ciencia ficción como detonador visual, narrativo y conceptual.

1. ¿Por qué trabajar futuros en el diseño?

Diseñar es actuar sobre el presente con efectos en el futuro. Por eso, las metodologías de futuros permiten expandir la creatividad, cuestionar inercias y generar escenarios que ayuden a pensar cambios sociales, culturales, materiales o tecnológicos.

Prospectiva Diseño especulativo Escenarios Crítica

La intención no es predecir el futuro, sino explorar posibilidades y construir visiones que alimenten la práctica del diseño.

2. Cono de plausibilidad

El Cono de plausibilidad es un modelo visual que organiza el espectro de futuros posibles, plausibles, probables y preferibles a partir del presente. Es una herramienta fundamental en estudios de futuros y diseño estratégico.

Diagrama del Cono de plausibilidad. Representa la expansión de posibilidades a medida que avanzamos desde el presente hacia futuros con distintos niveles de probabilidad.

¿Qué distingue cada zona?

  • Futuro probable: continuidad de tendencias.
  • Futuro plausible: podría ocurrir con ajustes intermedios.
  • Futuro posible: no contradice leyes naturales o sociales.
  • Futuro preferible: el que queremos construir.

El cono permite ubicar escenarios dentro de un rango inteligible, evitando confundir deseos con predicciones o especulaciones sin fundamento.

3. Ciencia ficción y diseño

La ciencia ficción funciona como laboratorio narrativo y visual para explorar futuros que aún no existen, pero que iluminan posibilidades, riesgos y dilemas del presente. En diseño, se utiliza para especular sobre tecnologías, experiencias, artefactos, sistemas sociales y transformaciones culturales.

Ilustración conceptual usada para explorar futuros mediante ciencia ficción. Las narrativas visuales ayudan a diseñar objetos, tecnologías y sistemas futuros.

¿Cómo se usa en diseño?

  • Para imaginar artefactos futuros mediante narrativa visual.
  • Para cuestionar sistemas actuales desde escenarios alternativos.
  • Para crear prototipos especulativos que generen debate.
  • Para explorar lo “posible” sin las restricciones del presente inmediato.

No se trata de hacer “fantasía futurista”, sino de usar la ficción como vehículo crítico para pensar otras formas de vida, objetos y relaciones.

4. ¿Cómo se usará en la clase?

En la EDINBA, estas metodologías se integrarán en actividades como:

  • Construcción de escenarios de futuros.
  • Proyectos especulativos a partir de ciencia ficción.
  • Análisis de plausibilidad para ubicar ideas dentro del cono.
  • Ejercicios narrativos y visuales basados en imágenes o prototipos.

Esto ayudará a que el diseño se entienda no solo como producción de objetos, sino como construcción de futuros posibles.
